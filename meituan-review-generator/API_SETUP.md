# 美团评价生成器使用指南

## 项目说明

本项目是一个美团评价生成器，完全在前端实现，不需要后端 API 或外部服务。评价内容基于预设模板生成，符合美团平台规范。

## 功能特性

- **六大商品类别**：冰洗、空调、电视、厨卫、个护、3C
- **六种评价风格**：
  - 真实自然型：客观中肯的评价
  - 服务夸赞型：突出服务体验
  - 高能好评型：积极但不夸张的评价
  - 吐槽转好评型：先略带不满，但最终整体肯定
  - 理性分析型：像科技博主一样分析产品
  - 家庭使用场景型：适合冰洗/空调/厨卫等家庭电器
- **智能模版池**：每个品类有6-8个开头模版、8个中段模版、8个结尾模版
- **随机拼接生成**：每次生成随机组合不同模版，确保评价多样性
- **商品名称集成**：自动将用户输入的商品名称插入评价中
- **图片上传**：支持商品图片预览（装饰性，无实际AI识别）
- **一键复制**：快速复制生成的评价内容

## 评价规范

生成的评价严格遵守以下规范：

✅ **允许内容**
- 客观描述商品特点、使用体验
- 真实的用户感受表达
- 合理的正面评价

❌ **禁止内容**
- 任何品牌名称（京东、淘宝、美团等）
- 极端词汇（最、顶级、神器、无敌等）
- 托评价词（"客服说"、"商家承诺"等）
- 虚假或夸张描述

## 技术实现

- **前端框架**：Next.js 16 (App Router)
- **样式框架**：Tailwind CSS
- **状态管理**：React useState Hook
- **生成逻辑**：模版池随机拼接算法
- **模版结构**：开头句 + 中段体验句 + 总结句
- **响应式设计**：适配桌面和移动设备

## 模版池设计

每个品类都包含三大模版池：

### 开头模版池 (6个)
- 时间/使用状态描述
- 产品表现引入
- 安装/到货体验
- 整体印象引入
- 外观特点描述
- 使用后的变化

### 中段模版池 (8个)
- 核心功能表现
- 使用体验细节
- 性能参数描述
- 设计特点说明
- 操作便捷性
- 噪音/耗电表现
- 安全/稳定表现
- 智能化功能

### 结尾模版池 (8个)
- 整体满意度
- 性价比评价
- 推荐意愿
- 购物价值
- 实用性总结
- 风格搭配
- 功能完整性
- 服务体验

### 生成算法
1. 随机选择开头模版
2. 随机选择中段模版
3. 随机选择结尾模版
4. 按"开头 + 中段 + 结尾"顺序拼接
5. 自动插入用户指定的商品名称

## 运行项目

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 `http://localhost:3000` 开始使用。

## API 功能说明

### 图片识别
- 支持 JPG、PNG 格式图片
- 自动识别物品类别、外观、颜色、使用场景
- 不识别品牌信息

### 评价生成
- 支持三种商品类别：冰洗、个护、3C
- 支持三种评价风格：真实自然型、夸赞服务型、高能好评型
- 生成的评价长度大于40字
- 严格遵守防违规规则

## 注意事项

- 确保 API 密钥安全，不要提交到版本控制系统
- API 调用有频率限制，请合理使用
- 如遇问题，请检查 API 密钥是否正确配置
